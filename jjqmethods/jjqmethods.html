Getter/Setter Paradime - you can get info ro set info thru jquery. can modify a group of dom elements without having to write a loop

.text() - gets all of the text contents and change the content, like $("h1").text("New Text!!!"); USE TEXT when manipulating user input. If you use HTML, they could inject code to take over your page

.html() - works like innerHTML - gets the FIRST contents and can change it like $("ul").html("I hacked your UL!!!");

.attr() - gets or sets a css attribute

.val() - gets or sets the value of an input or select .

.addClass() - adds a class to a dom element

.removeClass() - removes a class to a dom element

.toggleClass() - adds/removes a class to a dom element
